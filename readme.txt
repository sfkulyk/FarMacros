Edit.SelectionStartEnd.lua
	Works in Far Editor.
	Allows you to select block of text with two separate keypresses Ctrl+( and Ctrl+)
	Very useful if you need to select block during macro recording, or use F7 to find something specific.
 

ReloadMacros.lua
	Just reload all macros without restarting the FAR. Useful when you write/debug own macros.

SelectScroll.lua
	Select current item in current panel and scroll panel up for 1 item
